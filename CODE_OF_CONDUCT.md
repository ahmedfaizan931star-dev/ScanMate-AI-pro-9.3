# Code of Conduct

ScanMate AI Pro should remain a respectful, safe, and productive project.

## Expected Behaviour

- Be respectful and constructive.
- Focus criticism on code, UX, reliability, and security.
- Keep discussions clear and practical.
- Protect user privacy and avoid sharing sensitive data.

## Unacceptable Behaviour

- Harassment, insults, threats, or discriminatory language.
- Publishing secrets, keys, personal data, or private user files.
- Introducing hidden telemetry, spyware, or unsafe dependencies.

## Enforcement

Maintainers may close issues, reject pull requests, or block contributors who violate these expectations.
